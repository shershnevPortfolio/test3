# FRONTEND BOILERPLATE

## Install
```sh
$ git clone 
$ rm -rf .git
$ git init
$ npm install
```

## Develop
```sh
npm start       # for developing and   open  localhost:3000
npm run build   # for build. creating folder "build"
```
